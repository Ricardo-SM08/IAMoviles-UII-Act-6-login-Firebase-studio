# MySelftCat

## PROMPT
Actúa como un desarrollador Senior de Flutter. Necesito que programes un flujo de Login básico interactivo de 3 pantallas, sin conexión a base de datos, manejando la navegación a través de rutas nombradas (Named Routes). Entrégame el código completo de los 4 archivos necesarios para copiar y pegar.

1. Arquitectura y Estética:

El archivo principal será main.dart.

Los otros 3 archivos deben ir dentro de una carpeta llamada mispantallas y sus nombres serán: bienvenidos.dart, login.dart y signup.dart.

Estética OBLIGATORIA: Usa colores modernos y atractivos basados en una paleta morada (Lila #DCD0FF, Morado Principal #9C27B0, Morado Oscuro #4A148C y sombras Neón #E040FB). Los TextFields y botones deben tener bordes redondeados y fondos con relleno.

2. Pantalla 1: bienvenidos.dart (Ruta '/')

Debe contener una columna estrictamente dividida en 5 filas:

Fila 1: Título "Bienvenidos" en Morado Oscuro y tamaño grande.

Fila 2: Subtítulo "login desarrollado en flutter" en color gris.

Fila 3: Una imagen desde la red usando este enlace: https://raw.githubusercontent.com/Eliseo128/Imagenes-para-flutter-6to-I-fecha-11-feb-2026/refs/heads/main/bienvenida.png.

Fila 4: Un botón rectangular con bordes suaves que diga "Login". Al dar clic navega a la pantalla de login.

Fila 5: Un botón muy redondeado con sombra estilo neón que diga "Sign up". Al dar clic navega a la pantalla de signup.

3. Pantalla 2: login.dart (Ruta '/login')

Debe contener una columna estrictamente dividida en 7 filas:

Fila 1: Título "Login" en Morado Oscuro.

Fila 2: Subtítulo "inicia sesión con tu cuenta" en color gris.

Fila 3: Una etiqueta de texto "Email" en morado seguida de un TextField con fondo Lila claro.

Fila 4: Una etiqueta de texto "contraseña" seguida de un TextField oculto (obscureText).

Fila 5: Un botón redondeado que diga "iniciar sesión" con fondo Morado y sombra Neón.

Fila 6: Una fila que contenga el texto "no tienes cuenta? " y enseguida la palabra "Sign up" en negrita (que sea un botón/GestureDetector para navegar a signup).

Fila 7: Una imagen centrada usando este enlace: https://raw.githubusercontent.com/Eliseo128/Imagenes-para-flutter-6to-I-fecha-11-feb-2026/refs/heads/main/fondo.png.

4. Pantalla 3: signup.dart (Ruta '/signup')

Diseño libre en filas, pero debe contener:

Título "Sign up" y subtítulo "crear una cuenta, es gratis".

Etiquetas y TextFields estilizados para: Nombre de usuario, Correo electrónico, contraseña, y confirmar contraseña.

Un botón redondeado con el texto "Registro".

Al final, una fila con el texto "ya tienes una cuenta? " seguido de la palabra "Login" en negrita que regrese a la pantalla anterior.

Asegúrate de usar SingleChildScrollView en todas las pantallas para evitar errores de desbordamiento de píxeles (Overflow)."
